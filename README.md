# Bonam Maruthi Anushka | Personal Portfolio

Welcome to the repository for my personal portfolio website! This project is a responsive, aesthetically pleasing web application that showcases my skills, experience, projects, and achievements as a Computer Science Engineering student, Web Developer, and AI enthusiast.

## 🚀 Overview

This portfolio is designed to be a clean, modern, and interactive representation of my professional journey. It highlights my passion for blending computer systems architecture with elegant front-end designs, as well as my focus on AI-powered applications and advanced computational problem-solving.

### Features
- **Responsive Design**: Built to look great on desktop, tablet, and mobile devices.
- **Dynamic UI/UX**: Includes modern mesh gradients, smooth scrolling, and typing animations for an engaging user experience.
- **Project Showcase**: Highlights my latest work including "Kaamsetu", "Assignment Hub", and the "GDG Community Portal".
- **Skills Matrix**: A categorized display of my technical stack across Languages, Web Development, and Tools/AI.
- **Interactive Contact Form**: A functional form to get in touch for collaborations or internship opportunities.

## 🛠️ Tech Stack

This portfolio was built using core web technologies to ensure maximum performance and flexibility:
- **HTML5**: Semantic structure and accessibility.
- **CSS3**: Custom styling, Flexbox/Grid layouts, glassmorphism effects, and CSS animations.
- **JavaScript (ES6+)**: DOM manipulation, interactive components, typing effects, and form handling.
